NET_DEV_GetPictureSize

获取图片大小信息。

OS_BOOL NET_DEV_GetPictureSize(
    OS_INT32        lRealHandle,
    OS_INT32      * pWidth,
    OS_INT32      * pHeight
);

Parameters

lRealHandle
[in] 实时预览句柄，NET_DEV_StartRealPlay接口的返回值
pWidth
[out] 图片宽度
pHeight
[out] 图片高度

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
