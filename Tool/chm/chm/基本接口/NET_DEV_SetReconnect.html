NET_DEV_SetReconnect

设置是否自动重新连接，包括预览、布防。

OS_INT32 NET_DEV_SetReconnect(
    OS_BOOL        bEnable,
    OS_UINT32      dwInterval
);

Parameters

bEnable
[in] 是否使能自动重连，OS_TRUE-重连 OS_FALSE-不重连
dwInterval
[in] 重连间隔时间，单位：毫秒

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
