NET_DEV_Login

用户登录设备。

OS_INT32 NET_DEV_Login(
    char const             * pDeviceIP,
    OS_UINT16                wDevicePort,
    char const             * pUserName,
    char const             * pPassword,
    char const             * pRes
);

Parameters

pDeviceIP
[in] 设备IP地址
wDevicePort
[in] 设备端口号
pUserName
[in] 用户名
pPassword
[in] 密码
pRes
[in] 保留，置为NULL

Return Values

-1表示失败，其他值表示返回的用户ID值。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
