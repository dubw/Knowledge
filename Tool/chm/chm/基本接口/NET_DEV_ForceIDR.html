NET_DEV_ForceIDR

强制I帧，主码流动态产生一个I帧。

OS_BOOL NET_DEV_ForceIDR(
    OS_INT32                    lUserID,
    OS_INT32                    nChannelID,
    OS_INT32                    nStreamID
);

Parameters

lUserID
[in] 用户ID，NET_DEV_Login等登录接口的返回值
nChannelID
[in] 通道号，从1开始
nStreamID
[in] 码流号

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
