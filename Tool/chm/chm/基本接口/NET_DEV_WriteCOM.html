NET_DEV_WriteCOM

写串口数据，数据通道透传。

OS_BOOL NET_DEV_WriteCOM(
    OS_INT32                             lUserID,
    const NET_DEV_COM_DATA             * pData
);

Parameters

lUserID
[in] 用户ID，NET_DEV_Login等登录接口的返回值
pData
[in] 写串口数据信息

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
