NET_DEV_GetPlayBackPos

获取录像回放位置。

OS_BOOL NET_DEV_GetPlayBackPos(
    OS_INT32       lPlayHandle,
    OS_UINT32    * pPosition
);

Parameters

lPlayHandle
[in] 录像回放句柄，NET_DEV_PlayBack接口的返回值
pPosition
[in] 得到的播放进度位置，见下表
值 含义
0 正在播放
100 回放完成
>=200 回放过程中出现异常

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
