杰迈视讯SDK开发使用手册

NET_DEV_PlayBack

开启录像回放功能。

OS_INT32 NET_DEV_PlayBack(
    OS_INT32                    lUserID,
    NET_DEV_PALYBACK_PARA     * pPara,
    PLAY_DATA_CBF               fDataCallBack,
    void                      * pUser
);

Parameters

lUserID
[in] 用户ID，NET_DEV_Login等登录接口的返回值
pPara
[in] 录像回放参数设置
fDataCallBack
[in] 码流数据回调函数
pUser
[in] 回调函数参数

Callback Function

typedef void (SDKCALL *PLAY_DATA_CBF)(
    OS_INT32                lRealHandle,
    OS_UINT32               dwDataType,
    OS_UINT8              * pBuffer,
    OS_UINT32               dwBufSize,
    OS_VOIDPTR              pUser
);

Callback Function Parameters

lRealHandle
[out] 播放句柄，NET_DEV_PlayBack的返回值
dwDataType
[out] 数据类型，见下表
宏定义 值 含义
CALLBACK_STD_MEDIA_DATA 1 回调标准H264数据，此时pBuffer对应一个NET_DEV_STD_MEDIA_INFO类型的指针
CALLBACK_SREAM_INTERUPT 2 码流中断,若启用了重连,SDK会进行预览重连, 此时pBuffer = NULL
pBuffer
[out] 存放数据的缓冲区指针
dwBufSize
[out] 缓冲区大小
pUser
[out] 用户指针

Return Values

-1表示失败，其他值作为NET_DEV_StopRealPlay等接口的输入句柄。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
