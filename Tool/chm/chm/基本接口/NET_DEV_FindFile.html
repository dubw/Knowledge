NET_DEV_FindFile

查找录像文件。

OS_INT32 NET_DEV_FindFile(
    OS_INT32                   lUserID,
    NET_DEV_FIND_FILE_PARA   * pFindCond
);

Parameters

lUserID
[in] 用户ID，NET_DEV_Login等登录接口的返回值
pFindCond
[in] 查找录像条件参数

Return Values

-1表示失败，其他值作为NET_DEV_FindNextFile接口的输入句柄。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。

Remarks

不支持IPC。
