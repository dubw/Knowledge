NET_DEV_GetSDCardStatus

获取SD卡状态。

OS_BOOL NET_DEV_GetSDCardStatus(
    OS_INT32           lUserID,
    NET_DEV_DISK     * pSDCard
);

Parameters

lUserID
[in] 用户ID，NET_DEV_Login等登录接口的返回值
pSDCard
[out] SD卡状态信息

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
