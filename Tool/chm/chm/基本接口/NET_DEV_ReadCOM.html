NET_DEV_ReadCOM

读取串口数据，数据通道透传。

OS_BOOL NET_DEV_ReadCOM(
    OS_INT32                             lUserID,
    const NET_DEV_READ_COM_DATA_INFO   * Request,
    const NET_DEV_COM_DATA             * pData
);

Parameters

lUserID
[in] 用户ID，NET_DEV_Login等登录接口的返回值
Request
[in] 读串口数据的请求信息
pData
[out] 返回的数据内容

Return Values

成功返回TRUE，失败返回FALSE。接口返回失败请调用NET_DEV_GetLastError获取错误码，通过错误码判断出错原因。
