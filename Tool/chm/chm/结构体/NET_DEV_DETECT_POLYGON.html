NET_DEV_DETECT_POLYGON

多边形坐标信息。

typedef struct {
    OS_UINT8                 byValidPointNum;
    OS_UINT8                 byRes[3];
    NET_DEV_DETECT_POINT     points[MAX_POLYGON_POINTS];
} NET_DEV_DETECT_POLYGON;

Members

byValidPointNum
有效的角点坐标个数
byRes
保留位，置为0
points
具体角点坐标，取值范围：[0,MAX_POLYGON_POINTS]
