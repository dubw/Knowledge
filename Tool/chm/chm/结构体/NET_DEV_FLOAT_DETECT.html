NET_DEV_FLOAT_DETECT

漂浮物配置（输电线路防外破项目）。

typedef struct {
    OS_UINT8                 byEnable;
    OS_UINT8                 byFloatThreshold;
    OS_UINT8                 byRes[2];
    NET_DEV_DETECT_POLYGON   polygon;
    OS_UINT32                nMinArea;
    OS_UINT32                nFloatWindowLen;
} NET_DEV_FLOAT_DETECT;

Members

byEnable
是否使能检测，2-不使能 1-使能
byFloatThreshold
存在漂浮物的比例占多大判定为存在漂浮物, 取值范围：[1,100]
byRes
保留位，置为0
polygon
检测多边形坐标
nMinArea
面积小于该区域不作判断
nFloatWindowLen
漂浮物判定的时间窗口长度
