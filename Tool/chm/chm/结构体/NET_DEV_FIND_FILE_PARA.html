NET_DEV_FIND_FILE_PARA

查找文件参数。

typedef struct {
    OS_UINT16          wSearchCommand;
    OS_UINT16          wFileType;
    OS_UINT32          dwChannel;
    NET_DEV_TIME       struStartTime;
    NET_DEV_TIME       struStopTime;
    OS_UINT8           byRes[636];
} NET_DEV_FIND_FILE_PARA;

Members

wSearchCommand
查找命令，具体参数如下表
枚举名 含义
SEARCH_RECORD_FILE 查找文件
SEARCH_PICTURE 查找图片
wFileType
文件类型，0xffff-全部类型
dwChannel
通道号，从1开始
struStartTime
搜索时间段，起始时间
struStopTime
搜索时间段，结束时间
byRes
保留位，置为0
