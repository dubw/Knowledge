NET_DEV_VEHICLE_SERVER

服务器地址等信息（输电线路防外破项目）。

typedef struct {
    OS_UINT8                 byEnable;
    OS_UINT16                wPort;
    OS_UINT8                 byHeartbeatInterval;
    char                     sDeviceId[NET_DEV_DEVICE_ID_LENGTH];
    OS_UINT8                  byRes[15];
    char                      sIP[16];
} NET_DEV_VEHICLE_SERVER;

Members

byEnable
是否使能检测，0-不使能 1-使能
wPort
服务器端口号
byHeartbeatInterval
心跳间隔
sDeviceId
设备ID
byRes
保留位，置为0
sIP
服务器IP地址
