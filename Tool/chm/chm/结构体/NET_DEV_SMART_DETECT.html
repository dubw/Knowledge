NET_DEV_SMART_DETECT

设备越界侦测配置。

typedef struct {
    OS_UINT32                       dwEnable;
    OS_UINT32                       dwCheckTime;
    NET_DEV_SMART_DETECT_DATA       stSmartDetectInfo;
    NET_DEV_ALARM_HANDLE            struAlarmHandle;
    OS_UINT32                       byReserved[200];
} NET_DEV_SMART_DETECT;

Members

dwEnable
侦测是否使能，0-不使能 1-使能
dwCheckTime
检查告警状态间隔时间，单位：毫秒
stSmartDetectInfo
智能检测信息
struAlarmHandle
告警关联信息
byReserved
保留位，置为0
