NET_DEV_COM_DATA

串口透传数据结构体。

typedef struct {
    OS_UINT32          dwChannel;
    OS_UINT32          dwDataSize;
    OS_UINT8           byDataBuf[MAX_COM_TRANSPORT_DATA_LENGTH];
    OS_INT32           iReserved[8];
} NET_DEV_COM_DATA;

Members

nChannel
通道号，保留位，置为0
dwDataSize
串口数据长度
byDataBuf
串口数据存放数组
iReserved
保留位，置为0
