NET_DEV_ALARM_SCHEDULE

设备告警布防时间配置。

typedef struct {
    OS_UINT32               dwValidSegmentNum;
    NET_DEV_SCHEDLUE        sScheduleTime[NET_MAX_SCHEDULE_DAY][NET_MAX_SCHEDULE_SEGMENT];
    OS_UINT8                sReserved[400];
} NET_DEV_ALARM_SCHEDULE;

Members

dwValidSegmentNum
最大支持的布防时间段，只读
sScheduleTime
布防时间配置
sReserved
保留位，置为0
