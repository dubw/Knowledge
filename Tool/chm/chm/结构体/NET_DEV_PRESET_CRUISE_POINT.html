NET_DEV_PRESET_CRUISE_POINT

设备云台巡航参数。

typedef struct {
    OS_INT32                 nPreIndex;
    OS_INT32                 nStaySec;
    OS_INT32                 nMoveSpeed;
    OS_UINT8                 byRes[100];
} NET_DEV_PRESET_CRUISE_POINT;

Members

nPreIndex
预置点索引，取值范围：1~255
nStaySec
停留时间，单位:秒
nMoveSpeed
移动速度，取值范围：0~100，0-最慢 1-最快
byRes
保留位，置为0
