NET_DEV_STREAM_SHOW

设备显示OSD配置。

typedef struct {
    OS_UINT32               dwStreamID;
    NET_DEV_SHOW_TIMESTAMP  struTimeInfo;
    NET_DEV_SHOW_TEXT       struChanInfo;
    OS_UINT8                byRes[100];
} NET_DEV_STREAM_SHOW;

Members

dwStreamID
码流通道号，取值范围1~4
struTimeInfo
显示时间戳参数
struChanInfo
显示通道参数
byRes
保留位，置为0
