NET_DEV_ALARMOUT_CFG

告警输出配置。

typedef struct {
    OS_UINT32          dwEnable;
    char               sName[NET_NAME_LEN];
    OS_UINT32          dwNormalStatus;
    OS_UINT32          dwDelayTime;
    OS_UINT8           byRes[400];
} NET_DEV_ALARMOUT_CFG;

Members

dwEnable
是否使能，0-不使能 1-使能
sName
告警输出名称
dwNormalStatus
报警输出类型，0-断开 1-闭合
dwDelayTime
持续时间，单位：秒
byRes
保留位，置为0
