杰迈视讯SDK开发使用手册

NET_DEV_TIME

时间段信息。

typedef struct {
    OS_UINT16         wYear;
    OS_UINT8          byMonth;
    OS_UINT8          byDay;
    OS_UINT8          byHour;
    OS_UINT8          byMinute;
    OS_UINT8          bySecond;
    OS_UINT8          byRes;
} NET_DEV_TIME, *LPNET_DEV_TIME;

Members

wYear
年，如2015年
byMonth
月份，取值范围：[1,12]
byDay
日，取值范围：[1,31]
byHour
小时，取值范围：[0,23]
byMinute
分钟，取值范围：[0,59]
bySecond
秒，取值范围：[0,59]
byRes
保留位，置为0
