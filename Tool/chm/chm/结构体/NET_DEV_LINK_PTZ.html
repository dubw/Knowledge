NET_DEV_LINK_PTZ

设备PTZ联动方式配置。

typedef struct {
    OS_UINT8               byOperateCmd;
    OS_UINT8               byRes1[3];
    OS_UINT16              wOperateSeqNum;
    OS_UINT16              wPTZDelayTime;
    OS_UINT8               byRes[20];
} NET_DEV_LINK_PTZ;

Members

byOperateCmd
关联PTZ操作，0-没有 1-预置点 2-巡航 3-模式路径
byRes1
保留位，置为0
wOperateSeqNum
预置点、巡航、模式路径等操作序号
wPTZDelayTime
PTZ移动持续时间，单位：秒
byRes
保留位，置为0
